# Reflection

While building this math toolkit, I noticed that functions can make code easier to organize and reuse.

The most interesting part was using logic like the remainder operator (%) to check if numbers are even or odd.

One challenge was understanding how functions return values and how those values can be tested.

A toolkit like this could be used in calculators, educational programs, and other software that needs repeated calculations.

This assignment helped me practice writing, testing, and improving JavaScript functions.
